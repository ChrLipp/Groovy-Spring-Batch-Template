Groovy Spring Batch Template

Kickstart your batch development with Groovy

Download as .zip Download as .tar.gz View on GitHub

Get started

# you can skip cloning and download the .zip  or .tar.gz archive instead
git clone git://github.com/robokaso/Groovy-Spring-Batch-Template.git gsb
cd gsb

./gradlew run      # execute the sample job
./gradlew check    # run tests and generate reports
./gradlew distZip  # package the project for distribution
